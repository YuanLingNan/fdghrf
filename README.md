# 💬 语录随机生成器

一个简洁优雅的语录随机生成智能体，每天为你带来智慧与灵感。

## ✨ 功能特性

- 🎲 **随机语录** - 点击按钮即可获取随机智慧语录
- 📋 **一键复制** - 轻松复制喜欢的语录到剪贴板
- 📊 **生成统计** - 记录已生成的语录数量
- 🎨 **精美界面** - 渐变背景与毛玻璃效果
- 📱 **响应式设计** - 完美适配各种设备

## 🚀 快速开始

### 在线预览

直接打开 `index.html` 文件即可使用，无需安装依赖。

### 本地运行

```bash
# 克隆仓库
git clone git@github.com:YuanLingNan/fdghrf.git

# 进入目录
cd fdghrf

# 使用 Python 启动简易服务器（可选）
python -m http.server 8000

# 打开浏览器访问
# http://localhost:8000
```
